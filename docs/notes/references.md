## How to use this list

* If you’re after **neutralizing gas/plasma columns ahead of a cyclotron inflector**, start with **Linac4** and **HB2016** papers for measured neutralization vs pressure/species, then compare with **PXIE/IOTA** notes for electron-column concepts.
* For **compact focusing elements that intrinsically neutralize**, check the **Gabor lens** articles and the **AIP review**.
* If you’re exploring **aggressive current compression**, the **NDCX/NDCX-II** set shows what fully neutralized plasma channels can do.
